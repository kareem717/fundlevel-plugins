# Influencer Objection Handling

Draft replies to creators/influencers who push back on or need clarity about sponsorship pitches. You're offering creators paid work, not selling them a service.

## Context

FundLevel reaches out to influencers on behalf of brand clients. The typical first touch is casual and open-ended - no specific rate upfront, just expressing interest in working together on an upcoming video. Rates get discussed later once there's mutual interest.

## Core Philosophy

- Every objection is a smokescreen until you find the real reason
- Diagnose first, handle second
- Shorter is always better - creators skim
- Match the creator's energy
- Never bluff social proof - only name-drop brands you actually work with

This table grows as new objection types come in.

## When Drafting a Reply

1. Identify which objection type matches from the router above
2. Read the corresponding handle file in `references/`
3. Check Attio for the creator's history - past sponsorships, brands they've worked with
4. Match the creator's energy - casual if casual, formal if formal
5. Keep it SHORT. 2-3 sentences max. Every unnecessary line is a reason to not reply.
6. Cut anything that doesn't serve you. If a sentence provides no value and makes the email longer, delete it.

## Key Differences from Brand Objections

| | Brand Objections | Influencer Objections |
|---|---|---|
| **You're selling** | FundLevel's service (sourcing, production, scale) | Paid work (content they already make) |
| **Reply length** | 3-5 sentences | 2-3 sentences |
| **Tone** | Professional-casual | Casual-friendly |
| **Close** | "Cool if I shoot over a profile?" | Diagnostic question or direct ask for rates |
| **Name-dropping** | Drop brand clients (PostHog, TigerData) | Drop brands the creator would want to work with |
| **Urgency** | Low - plant seeds | Zero - creators hate pressure |
| **Goal of the reply** | Keep the door open + propose next step | Diagnose the real reason + keep convo going |

## Universal Rules

- Shorter is always better. Creators skim. Every extra sentence is a reason to not reply.
- Never mention rates in an objection reply unless you're explicitly asking for theirs.
- Never accept a surface-level no. Always dig for the real reason.
- Name-drop brands that are cool/desirable in the creator's specific niche.
- Never name-drop a brand the creator has already worked with - there's no pull there.
- "Going forward" signals you know they've done sponsorships before. Use it.
- If a creator has a manager/agent, pivot to them. Don't go around representation.
- Cut ruthlessly. If a line provides no value for you AND makes the email longer, it shouldn't be there.
- Never say anything that implies you'll spam them.
- No em dashes - use regular dashes.
